# **App Name**: Odisha Waste Watch

## Core Features:

- Role-Based Access Control: Secure access control based on roles: Admin, Civilian (Transport Personnel), and GP/ULB.
- Interactive Organizational Structure Map: An interactive map displaying the hierarchical structure of districts, blocks, GPs, and tagged ULBs with visual indicators for activity and waste transport status.
- Driver & Vehicle Tracking: Real-time tracking of drivers and vehicles, including location, assigned GP, destination ULB/MRF, route, and status.
- Invoice Auto-Extraction: AI-powered tool to automatically extract information (GP name, ULB name, vehicle details, waste quantities) from uploaded waste invoice images, with a manual correction option.
- Waste Reconciliation Dashboard: A dashboard to reconcile waste sent from GPs versus waste received at ULBs, highlighting discrepancies and providing aggregated weekly, monthly, and annual data.
- Meeting Scheduler: Schedule district/block/ULB meetings, add participants and agenda, and upload minutes of meetings and photographs.
- Waste Collected Summary: A summary for Civilian module showcasing a Segregation summary showing the net quantity per waste type and presenting a pie chart visualization.

## Style Guidelines:

- Primary color: Forest Green (#38761D) to reflect environmental sustainability and the Government of Odisha's branding.
- Background color: Light Beige (#F2F1E6) to create a clean and professional government UI theme.
- Accent color: Slate Gray (#708090) for CTAs and important UI elements to ensure visibility without being overly distracting.
- Body and headline font: 'PT Sans', a modern and readable humanist sans-serif for both headings and body text.
- Use clean and simple icons that visually represent waste types, locations, and actions.
- Employ a dashboard-oriented layout with clear sections, consistent spacing, and intuitive navigation.
- Subtle animations for loading states and data updates to enhance user experience.
